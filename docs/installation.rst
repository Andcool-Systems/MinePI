Installation

Stable
~~~~~~

^^^^^^^^^

^^^^^^^^^

    pip install MinePI -U

^^^^^^^^^^^
From Github
^^^^^^^^^^^

.. prompt:: bash

    pip install git+https://github.com/benno1237/MinePI.git#egg=MinePI
